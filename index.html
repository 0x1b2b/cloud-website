Getting Started

Welcome to Azure Architecture Designer! This tool helps you create professional Azure cloud architecture diagrams with ease.

Quick Start Guide

  1. Add Components: Drag Azure services from the left panel onto the canvas
  2. Resize & Arrange: Use resize handles or the properties panel to adjust size and position
  3. Group/Ungroup: Select multiple components and group/ungroup for easier management
  4. Use Layers: Organize your architecture into layers
  5. Analyze: Use the Cost Calculator and Security Analysis for insights
  6. Export: Save your diagram as a screenshot or JSON file

Frequently Asked Questions

Is Azure Architecture Designer free?

Yes, the basic version is completely free to use. We offer premium features for enterprise users.

Can I export my diagrams?

Absolutely! You can export as high-resolution PNG screenshots or JSON files for sharing and collaboration.

Do you support other cloud providers?

Currently, we focus on Azure services. We're considering adding support for AWS and Google Cloud in the future.

How do I save my work?

Your work is automatically saved to your browser's local storage. You can also export your designs as JSON files.
